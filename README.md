# Synapse AI

A production-grade AI chat application built with MERN stack.

## 🚀 Features
- Real-time AI chat (streaming)
- Authentication (JWT)
- Chat history (MongoDB)
- Multi-conversation support

## 🛠 Tech Stack
- React (Frontend)
- Node.js + Express (Backend)
- MongoDB (Database)
- OpenAI API
